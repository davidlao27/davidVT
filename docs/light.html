Lighting


Lighting is key for some backgrounds.. Sunny day? Nighttime?

You should read the menu section before you read next: on the Others section, the first textbox specifies an hexadecimal "RGB" or "RRGGBB" (WITHOUT THE #) for the background color to use. The default is 0F0 standing for "Full Green".

The second textbox specifies the URL for a background image that overrides the background color (which by the way is usually used for chroma) to be replaced directly by a perfect chroma-filtered background image.
